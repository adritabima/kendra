# My Webpage (Starter)

This is a minimal static webpage scaffold (HTML/CSS/JS).

Quick start:

1. Open `index.html` in your browser (double-click the file).
2. Or run a simple local server and visit http://localhost:8000:

```bash
python -m http.server 8000
```

Next steps:
- Edit `index.html` to add content
- Update `styles.css` for layout and theme
- Add more interactivity in `script.js`
